# Poker History Stats

Локальное приложение для импорта архивов с историей турниров GG Poker и просмотра статистики через браузер.

Что умеет:

- импортирует ZIP-архивы с текстовыми файлами турниров;
- хранит общую историю в SQLite и не дублирует уже загруженные турниры;
- показывает, как часто вы занимаете каждое место;
- фильтрует статистику по бай-ину и призовому фонду;
- умеет добавлять новые архивы через веб-форму.

## Запуск

```powershell
python app.py
```

После этого откройте `http://127.0.0.1:8000`.

При старте приложение автоматически импортирует все ZIP-архивы из корня этой папки.

## Ручной импорт

```powershell
python app.py import . --recursive
```

## Где лежат данные

- база статистики: `data/poker_stats.sqlite3`
- загруженные через веб архивы: `data/uploads/`
